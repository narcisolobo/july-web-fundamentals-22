RULE!!! THE PARENT TELLS THE CHILDREN HOW TO BEHAVE!!

  1. Identify the elements that must be side-by-side
  2. Find their direct common parent
  3. If there is no direct common parent, you must create one
  4. Apply display: flex to the parent
  5. Add the other amigos: justify-content: space-between; align-items: center
