Rule #1 of Position: DON'T ABUSE POSITION

A good time to use position is when an element breaks the plane of another element.

All values of the position property are relative not just the relative value, it just becomes a question of what they are relative to.

CSS Position

Position Static

position: static

Default value of position.

Position Relative

position: relative

Element is positioned relative to its normal position.

Position Absolute

position: absolute

Element is positioned relative to its closest positioned ancestor.

Position Fixed

position: fixed

The element is positioned relative to the scroll position.
