Intro to the DOM

Document Object Model - A model of the document as an object.

An Object is reference datatype with key: value pairs.

An object IRL has characteristics and things it can do. For instance, a car has a make, a model, number of wheels, a color, etc. These are the characteristics of the car. A car can also drive, it can also beep, etc. These are things a car can do.

In JS - we call these characteristics properties.

And we call things an object can do methods. A method is just a function that belongs to an object.
